Developer Note

To activate email sending, follow the 3-step setup below.

  1. Go to emailjs.com → create a free account
  2. Add an Email Service (Gmail) → copy your Service ID
  3. Create an Email Template with variables:
    {{from_name}}, {{from_email}},
    {{subject}}, {{message}} → copy your Template ID
  4. Copy your Public Key from Account → API Keys
  5. In contact.html, replace these 3 values:
    • user_REPLACE_WITH_YOUR_KEY → your Public Key
    • YOUR_SERVICE_ID → your Service ID
    • YOUR_TEMPLATE_ID → your Template ID

Free tier: 200 emails/month. No backend needed.
